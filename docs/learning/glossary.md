# Glossary

This glossary gives short beginner-friendly meanings for the main terms used in the repo.

## Base pair

A paired position in double-stranded DNA.

## Bead

A coarse-grained interaction site representing a group of atoms.

## Coarse-grained

A lower-detail molecular representation where multiple atoms are grouped together.

## Complex

A molecular assembly made of multiple pieces, such as protein plus DNA.

## dsDNA

Double-stranded DNA.

## Force field

A set of rules and parameters describing how a simulation model behaves.

## Hand-off bundle

A packaged set of files prepared for later computational work, such as HPC simulation setup.

## Helicase

A protein machine that interacts mechanically with nucleic acids such as DNA.

## Hexamer

A six-subunit assembly.

## HPC

High-performance computing. In practice, this usually means running larger jobs on more powerful shared compute infrastructure.

## Ion

A charged particle, such as sodium, chloride, or a divalent species.

## Martini

A family of coarse-grained molecular force fields.

## PDB

A common structural file format used for biomolecular coordinates.

## Protein topology

A file or set of files describing what particles exist in the model and how they are connected or typed for simulation.

## Rescue branch

In this repo, the practical fallback workflow used to complete the delivery when the originally intended path did not work cleanly.

## Solute

The main molecular object of interest in the environment, here mainly the protein + DNA complex.

## Solvent

The surrounding medium, here represented mainly by water beads and ions.

## Topology

A file describing molecular identities, connectivity, and supporting force-field structure beyond simple coordinates.

## Twinkle

The ring-like helicase protein complex being modeled in this repo.
